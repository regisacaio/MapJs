A simple way to work with Maps

Made with Love for Padawans

The MapJs gives to you a new perspective for work with Maps in web, mobile or other plataforms using Javascript. To use, it's simple and faster. Just add js source in HTML page and start. 
      
      <script type="text/javascript" src="MapJs.js"></script>
      <script type="text/javascript">
       Map.getMap(#element");
      </script>

How I use? (Guide)

To start the Frame, just follow the first topic. For extract and understand all functions, keep reading.

Get Map from my actual location
      
       Map.getMap("#element");
PS: You can change width and height map, adding the parameters.



Get my actual Coordinates 
      
       Map.getLocation("#element");
